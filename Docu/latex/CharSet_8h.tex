\subsection{Detailed Description}
Defines the \hyperlink{structCharSet}{Char\-Set} data structure. \hyperlink{structCharSet}{Char\-Set} can be initialized with all the letters of the english alphabet and allows extraction of random letters.

It's implemented as a singly linked list. Each node contains a letter and a random I\-D number. When it's initialized, the set is populated with all letters of the alphabet and sorted by these random I\-D numbers, randomizing it.

Extracting a random letter is done by extracting the head of the list.
