\subsection{Detailed Description}
Defines the \hyperlink{structWordSet}{Word\-Set} data structure. \hyperlink{structWordSet}{Word\-Set} allows insertion of strings and consulting wether a string is contained or not.

It's implemented with a trie. Each node contains an array of children (pointers to trie nodes).
